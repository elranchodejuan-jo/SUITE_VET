# Carpeta de Libros de SUITE VET

Coloca en esta carpeta todos tus archivos PDF de libros de estudio que quieras integrar en la biblioteca digital del proyecto.

### Instrucciones:
1. Copia y pega aquí tus archivos `.pdf`.
2. Para facilitar la vinculación, procura usar nombres simples y en minúsculas, por ejemplo:
   - `guyton_fisiologia.pdf`
   - `farma_veterinaria.pdf`
   - `microbiologia_manual.pdf`

Una vez que los coloques aquí, avísame y yo me encargaré de mapearlos en la base de datos de referencias y crear sus accesos de descarga directa.
